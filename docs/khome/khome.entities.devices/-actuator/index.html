An Actuator holding entity state and attributes

In Khome, the Actuator is the mutable representation of an entity in home assistant.

Parameters

S

the type of the state object that represents all mutable state values of the entity. Has to implement the State interface.

A

the type of the attributes object that represents all immutable attribute values of the entity. Has to implement the Attributes interface.
